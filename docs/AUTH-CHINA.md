# 中国大陆登录方案说明

## 结论（先说清楚）

| 方式 | Supabase 能直接勾选吗 | 个人开发者难度 | 建议阶段 |
|------|----------------------|----------------|----------|
| 邮箱 | ✅ 已接好 | 低 | 现在就能用 |
| 手机号 + 验证码 | ⚠️ 需接短信服务商 | 中 | **短期最现实** |
| 微信扫码登录 | ❌ 需微信开放平台 | 高（通常要企业） | 有营业执照后 |
| 抖音扫码 | ❌ 需字节开放平台 | 很高 | 不建议先做 |

**微信 / 抖音登录不能**像在 Supabase 里打开 Email 开关一样「一键完成」，要单独申请开放平台、一般还要**企业主体**和备案网站。

---

## 为什么微信扫码不简单

1. 去 [微信开放平台](https://open.weixin.qq.com/) 注册**网站应用**
2. 多数情况需要 **企业认证**（约 300 元/年量级，且要营业执照）
3. 配置授权回调域名（通常要求 **ICP 备案** 的域名）
4. 用 AppID + AppSecret 做 OAuth，后端换 `access_token`（**Secret 不能放前端**）
5. 和 Supabase 用户体系打通（自定义登录或迁移到国内身份服务）

个人、无公司：微信**网站扫码登录**往往卡在认证这一步。

---

## 短期推荐（大陆用户）

### 方案 A：手机号 + 短信验证码（优先推荐）

- 用户习惯好，个人项目相对可行
- 短信：阿里云 / 腾讯云（按条计费，约几分钱一条）
- 实现：Vercel Serverless Function 或 Supabase Edge Function 发码 + 校验，再创建 Supabase 用户

### 方案 B：保留邮箱 + 微信内分享 H5（不做微信登录）

- 先靠邮箱或游客练习获客
- 课程分享到微信里打开链接（仍是邮箱注册或免登录试玩）

### 方案 C：国内身份云服务（省开发）

- [Authing](https://www.authing.cn/)：支持手机、微信、邮箱，有免费档
- 适合「主要大陆、不想自己接微信 OAuth」
- 需要把现在 Supabase Auth 逐步迁到 Authing，或 Auth 用 Authing、数据仍放 Supabase

---

## 中期（有企业资质后）

1. 注册微信开放平台 → 网站应用 → 微信扫码登录  
2. 域名备案（rootgrammar.com 或改用 .cn + 国内托管）  
3. 后端接口用 AppSecret 换 openid，绑定到 `auth.users` 或自建 `profiles` 表  

抖音登录同理，需[字节跳动开放平台](https://open.douyin.com/)，门槛类似，**不建议作为第一个社交登录**。

---

## 和当前 Root Grammar 的关系

- 你已跑通：**邮箱 + Supabase + 学习进度表** → 可继续给海外或能收邮件的用户用  
- 大陆主推前：**先定登录方式**（手机验证码 vs 等微信 vs Authing）  
- 练习页是 **PC 键盘造句**，大陆用户仍可在电脑浏览器用；登录方式与产品形态不冲突  

---

## 建议路线（一个人）

1. **本周**：邮箱登录上线 + Vercel 环境变量 + 大陆朋友试玩  
2. **下一步**：加 **手机号验证码**（若你决定做大陆主市场）  
3. **有公司后**：再申请 **微信扫码**  
4. **暂不做**：抖音登录  

若选定「手机号」或「Authing」，在 Cursor 里说一声，可按选定方案改代码。
